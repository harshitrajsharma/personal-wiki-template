# Raw Sources

Put unprocessed source material here.

Use:

- `raw/inbox/` for new files waiting to be processed.
- `raw/archive/` for processed material you still want to keep.
- `raw/references/` for source files you often return to.

Rules:

- Raw files are treated as immutable sources.
- The AI assistant should read from this folder, then write cleaned notes into `wiki/`.
- Do not put credentials, identity documents, tokens, or private secrets here.
