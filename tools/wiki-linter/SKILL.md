# Wiki Linter

Use this skill when the user asks to review, clean, audit, or lint the wiki.

## Check

- Broken links.
- Empty important sections.
- Pages without frontmatter.
- Notes placed in the wrong folder.
- Stale project statuses.
- Old tasks that should be closed, moved, or reviewed.
- Raw files that have not been processed.

## Output

Give the user:

- high-priority fixes
- low-priority cleanup
- suggested next actions

Do not delete anything without explicit permission.
