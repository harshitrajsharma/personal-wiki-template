# Tools And Skills

This folder contains reusable instructions for AI assistants.

Each skill is a small markdown file that explains how the assistant should perform a specific workflow.

Included skills:

- `note-manager`: ingest and organize raw notes.
- `structure-enforcer`: choose the correct wiki location before creating files.
- `wiki-linter`: audit the wiki for broken links, stale pages, and misplaced files.
- `daily-planner`: build or refresh `wiki/today.md`.

These are instructions, not required software dependencies. Any AI assistant can read them and follow the workflow.
