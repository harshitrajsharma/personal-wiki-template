# Daily Planner

Use this skill when the user asks to plan the day, reset today, or choose next actions.

## Inputs

Read:

- `wiki/today.md`
- `wiki/entities/core/goals.md`
- `wiki/entities/core/dashboard.md`
- active project pages in `wiki/entities/projects/`

## Workflow

1. Identify today's realistic capacity.
2. Choose the few tasks that matter most.
3. Preserve unfinished tasks that are still relevant.
4. Write the plan into `wiki/today.md`.
5. Keep the plan short enough to execute.

## Daily Plan Shape

Use:

- focus
- top 3 tasks
- schedule blocks if useful
- notes
- end-of-day review
